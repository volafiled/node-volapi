volapi in node
===

Ever wanted to make a volafile bot, but python isn't your thing and Dongo's
elixir is for plebs?
`volapi` to the rescue!

Installation
---

You know the drill:

```shell
npm install volapi
```

API
---

You got a `Room` and it emits `Message`s and `File`s and other stuff!

See sample_client.js
